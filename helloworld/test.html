Обчислення суми багатьох чисел

Сума чисел від 1 до 100 

 // Ініціалізуємо акамулятор
s=0;

 // Для всіх чисел від 1 до 100
for(i=1; i<=100; i++) {
  // Додаємо число до акамулятора
  s += i;
}

Сума:

s=
